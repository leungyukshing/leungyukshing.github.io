Java常用输入输出

Posted on 2018-03-14 Edited on 2021-06-22 Views: Valine:

 在初学java的时候，我们总是比较关心一个程序如何从命令行读取输出，如何输出数据。在这篇post，我就来给大家讲解一下关于java的一些常用输入输出操作。

输入

 在java中的输入同样和C++不一样，是使用本身提供的Scanner类来实现的。我们先来看一个简单的程序ComputeAreaWithConsoleInput.java：

 从上面的程序可以看出，我们经常使用的是这条语句从命令行读入用户输入的数据：

Scanner input = new Scanner(System.in);

 这条语句的意思是，创建了一个类型为Scanner的对象input。我们可以通过调用这个input的对象的各种方法进行读入数据。

Methods for Scanner Objects：

Method Description
nextByte() reads an integer of the byte type
nextShort() reads an integer of the short type
nextInt() reads an integer of the int type
nextLong() reads an integer of the long type
nextFloat() reads a number of the float type
nextDouble() reads a number of the double type
next() reads a string that ends before a whitespace character
nextLine() reads a line of text(i.e., a string ending with the Enter Key pressed)

输出

 相比起输入，输出就稍微简单一下，上面程序中有两条输出语句，这就给出了我们常用的一个输出模板：

System.out.print(String str);
System.out.println(String str);

 其中，第一条的输出后是没有换行的，第二条输出后是有换行的。参数str是String类型的变量，可以直接用’+’连接。

 至此，java程序中最基本的输入输出语法就介绍完了，谢谢！
